id = 변수

객체(object)에 대해서 잘 이해 해야함.
객체 == 속성 + 메소드
    ㄴ  속성 == 변수(속성은 변수의 일종이다.) == 변수{?, 속성, ??}
        메소드 == 함수(메소드는 함수의 일종이다.) == 함수{?, 메소드, ??}

id를 사용한다 = 재사용한다.

document.getElementById("test").innerHTML = z;
test라는 id를 가진, 요소를 찾고. innerHTML로 해당 콘텐츠 내용을 저장한다.

[getElementById]
id = test;
fopen_s(&fp, test, "w");

[innerHTML]
innerHTML = z;
fprintf(fp, %s\n, innerHTML);
